# Ressources YouTube

Dépôt centralisant l'ensemble des ressources (présentations, supports de formation, documents) liées aux vidéos YouTube. Chaque thématique a son propre sous-dossier.

## Structure

```
Ressources Youtube/
└── Formation complete Kobotoolbox/
    ├── KoboToolbox_Intro.pptx
    ├── KoboToolbox_Module1.pptx
    ├── ...
    └── KoboToolbox_Module12.pptx
```

## Thématiques disponibles

- **Formation complete Kobotoolbox** — Formation complète sur KoboToolbox (introduction + 12 modules)

## Ajouter une nouvelle thématique

1. Créer un nouveau sous-dossier à la racine, nommé selon la thématique
2. Y déposer les fichiers correspondants
3. Mettre à jour la liste des thématiques ci-dessus
4. Commit et push :
   ```bash
   git add .
   git commit -m "Ajout de la thématique X"
   git push
   ```
