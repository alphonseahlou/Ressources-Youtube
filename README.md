# Ressources YouTube

Dépôt centralisant l'ensemble des ressources (présentations, supports de formation, documents) liées aux vidéos YouTube. Chaque thématique a son propre sous-dossier.

## Structure

```
Ressources Youtube/
└── Formation complete Kobotoolbox/
    ├── KoboToolbox_Intro.pptx
    ├── KoboToolbox_Module1.pptx
    ├── ...
    └── KoboToolbox_Module12.pptx
```

## Thématiques disponibles

- **Formation complete Kobotoolbox** — Formation complète sur KoboToolbox (introduction + 12 modules)

## Règles d'utilisation

- **Usage** : ces ressources sont fournies à titre de support pédagogique pour accompagner les vidéos YouTube associées. Elles sont destinées à un usage personnel ou éducatif.
- **Citation de la source** : si tu réutilises ou partages un contenu, mentionne la chaîne YouTube d'origine et/ou ce dépôt.
- **Pas de modification du contenu original sans le signaler** : si tu adaptes une présentation, indique clairement qu'il s'agit d'une version modifiée.
- **Pas de redistribution commerciale** sans autorisation préalable de l'auteur.
- **Mise à jour des fichiers** : ne pas écraser un module existant avec une version incomplète — créer une nouvelle version (ex: `_v2`) si le contenu n'est pas définitif.
- **Formats** : privilégier des formats ouverts ou largement supportés (`.pptx`, `.pdf`, `.md`) pour assurer la compatibilité à long terme.
